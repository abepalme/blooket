Notes:
Requires Chrome or Chromium-based browser for a computer. It will not work for mobile/tablet devices.
If you have a computer and you want it on your tablet, you can sync your bookmarks into the Chrome app for iPadOS and AndroidOS.


How to install Blooket cheats:

1) Download Bookmarklets.html file.
2) Go to chrome://bookmarks/
3) Click the three dots in the top right corner.
4) Import bookmarks.
5) Click select file.
6) It should save to your bookmarks bar.
7) Rename main folder to "Blooket"

How to use:

1) Click folder in Bookmarks Bar.
2) Click subfolder, then find the one you want.
3) It might throw an error, but just hit 'OK' and it will ignore it and execute the bookmarklet.
